Claro! Aquí tienes un resumen de los tamaños y proporciones recomendadas para las imágenes en las pantallas que hemos diseñado, para que puedas crear tu manual de usuario:

Para asegurar la mejor visualización en Detodoya.com, considera las siguientes proporciones y tamaños para tus imágenes:

Imágenes de Categorías (Carrusel de la Página de Inicio):

Proporción: Preferiblemente cuadradas o ligeramente horizontales (como 4:3).
Tamaño recomendado: Aproximadamente 300x280px (para miniaturas) hasta 800x600px o similar para mayor calidad. La prioridad es la proporción.
Importante: Asegúrate de que las imágenes tengan buena resolución para que se vean nítidas en diferentes tamaños de pantalla.
Imágenes de Productos (Grid del Catálogo y Productos Destacados en Inicio):

Proporción: Vertical (como 2:3 o 3:4).
Tamaño recomendado: Para las tarjetas de producto, idealmente imágenes con una altura considerable, por ejemplo, 480px de alto (con un ancho proporcional). Para una buena calidad, piensa en resoluciones como 600x900px o 800x1200px.
Consideración: Estas imágenes deben llenar bien el espacio vertical de la tarjeta para un aspecto uniforme en el grid.
Imágenes Principales de Producto (Página de Detalle de Producto):

Proporción: Variable, se adaptan para mostrar el producto. Pueden ser cuadradas (1:1), verticales (2:3, 3:4) u horizontales (4:3).
Tamaño recomendado: Lo más grande y de alta resolución posible, manteniendo una carga rápida. Por ejemplo, hasta 1000x1000px o 1200x800px.
Importante: El sistema está diseñado para mostrar la imagen principal de forma que "contenga" la imagen dentro de un espacio, por lo que una alta resolución es clave para el zoom.
Miniaturas de Galería (Página de Detalle de Producto):

Proporción: Vertical (como 2:3 o 3:4).
Tamaño recomendado: Aproximadamente 120x188px para miniaturas, pero asegúrate de que sean versiones más pequeñas de las imágenes principales para una carga eficiente.
Consideraciones Generales:

Videos: Para cualquier sección donde se admitan videos (historias, imágenes de producto), usa un formato estándar (MP4 preferiblemente) y una resolución que equilibre calidad y tamaño de archivo (por ejemplo, 720p o 1080p).
Optimización: Siempre se recomienda optimizar las imágenes para la web (compresión sin pérdida de calidad) para asegurar tiempos de carga rápidos.
Coherencia: Mantener una proporción y estilo visual coherente en cada tipo de imagen (categorías, productos) ayudará a que tu catálogo se vea más profesional y atractivo.
